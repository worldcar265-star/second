# 원본

여기는 사용자가 직접 입력한 자료들이 쌓이는 곳입니다.

> **Claude는 이 폴더의 파일을 수정·삭제하지 않습니다 — 읽기만.**
> 원본은 변하지 않는 진실. 위키는 그 위에서 자라는 해석.

## 하위 폴더

자료 종류별로 자라난다 (필요해질 때 만들어진다):

- `일기/` — 날짜별 자유 기록 (예: `2026-04-30.md`)
- `대화/` — Claude와의 의미있는 대화 정리본
- `미디어/` — 사진·음성·영상
- `외부/` — 외부에서 가져온 자료 중 자기와 관련된 것 (좋아하는 인용, 자신을 떠올리게 한 기사 등)

폴더가 아직 안 보이면, 첫 자료를 추가할 때 만들어집니다.

## 작성 팁

- 일기 파일은 `YYYY-MM-DD.md` 형식 권장 (정렬·검색 쉬움)
- 자유롭게 써도 됨 — Claude가 읽고 정리해 위키에 옮긴다
- 형식 신경 쓰지 말 것. 거친 메모도 충분
