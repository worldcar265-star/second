# 향미유 R&D 의뢰 케이스 + 사실 기반 응답 규칙

## 박준호 자기 진술 원문 (응답 규칙)

> "항상 사실에 입각해서 출처를 찾아서 대답을 해줬으면 좋겠고 너의 추측이나 유추했던것을 주석이라던지 어떤 출처에 근거해서 이런 생각이 들었다고 논리를 나에게 같이 설명해주도록해줬으면 좋겠어"

## 박준호 의사결정·진술 원문 (대화 흐름순)

- "라멘에 넣는 기름중에 치유가 뭐얖" → 향미유 카테고리 탐색 시작
- "이 기름들을 rnd개발 의뢰가 들어왔어. Oem 생산공장들이 많지않은것같은데 서치해줘" → 프랜차이즈 유통사로부터 RnD 의뢰 수주
- "내가 만들면 리스크를 내가 지는데 그 의뢰를 받은게 개발인데 이 공급리스크를 줄이기위한 대안" → 공급 리스크 분리 의지 명시
- "지금 프렌차이즈 유통사이고 공장리스트업까지만 해줘도되고 제조해서 납품해줘도 된다고하시네 월 사용량도 금요일에 보여주시기로햇어" → 의뢰처 정체·옵션·미팅 일정 확인
- "현재 oem해놓고 쓰고잇는데 가격이 너무비싸서 새로 하려고하는것같아" → 의뢰처가 기존 OEM 보유, 단가 인하 동기
- "내가 생각하는건 그래서 타당성 분석부터 하는게 맞다고 보거든 그래야 내가 뭐 하나 개발당 얼마를 받던지 할 수 잇어서" → 타당성 분석 단계 분리 발상
- "업계표준이 개발해주고 하는데 250-500 이라하더라고 500은 한개만 할때 여러개 할땐 많이 깎아서 250정도라더라고" → 업계 단가 정보 (박준호 출처)
- "타당성분석을 그냥 상담하듯이 대화로 하고" → 무료 상담 + 유료 착수 분리 프레임
- "머리속 시물레이션 아니고 클로드 켜놓고 너에게 내가 값을 입력하면 바로 출력하게 준비" → 미팅 중 클로드 실시간 활용 결정
- "원하는 향미유는 3가지: 라유, 치유, 범용 향미유 — 난 이걸 양파 향미유로 생각하고 잇어. 범용 향미유는 라멘에도 쓰지만 여기 업체에서 미역국과 소고기무국에도 넣을 향미유도 의뢰를햇엇는데 한번에 해결가능할거같아서" → 양파 범용 향미유 발상
- "내가 생각하는 방식은 치유는 좀 힘들것같은데 범용 향미유에 라유팩을 우리면 라유가 되는 방식이 향미유 하나에 가루방식을 더하면 좋을것같아서" → 베이스 + 향미팩 시스템 발상
- "라유는 향신료를 우리는게아미라 향신채잖아" → 향신료 vs 향신채 구분 통찰 (R&D 핵심 분기점)
- "야채믹스를 얼려서 녹인후 매장에서 우리는건? 그거 만드느니 라유를 만드는게나으려나" → 자기 발상의 단점 메타인지로 검증
- "아재꼬막에 즉판 사업자가 있긴있어" → 즉판 사업자 자격 보유 확인
- "온라인택배판매 가능한걸로 알고잇는데" → Claude의 잘못된 정보 정정 (즉판도 통신판매 가능)

## Claude 응답 중 박준호가 동의·인용한 부분

- 즉판 사업자는 온라인 택배 판매 가능 (Claude의 처음 답변 정정 수용)
- 라유는 향신채(고추)를 우리는 것 — 가루팩으로 100% 재현 어려움
- 야채믹스 냉동 방식보다 라유 액상 OEM이 경제성 압승
- 타당성 분석을 무료 상담으로 진행, 착수는 유료로 명확히 분리

## Claude 응답 중 출처 없이 던졌던 부정확한 정보 (박준호가 짚어서 정정됨)

- "일본 라멘 체인 70%가 향미 페이스트 방식 사용" → 출처 없음, 정정 완료
- 한국 라멘 프랜차이즈 시장 규모 추정치 → 검증 안 됨
- 매장당 월 치유 사용량 추정치 → 출처 없음

## 부속 갱신 (대화 중 도출된 추가 사실들)

### 사업 라인업 갱신
기존 양념장 라인에 더해 **향미유 라인** 추가 검토 중:
- 치유 (닭기름 향미유)
- 라유 (매운 고추 향미유)
- 양파 범용 향미유 (라멘·미역국·소고기무국 통합)

프랜차이즈 유통사로부터 R&D 의뢰 수주 (2026-05) — 의뢰처 케이스가 azae 향미유 라인업의 첫 레퍼런스가 될 가능성.

### 사업 자격·허가 현황
향미유 양산 시 필요 자격:
- **식품유통전문판매업 신고** — 일반 식품 OEM 위탁 후 유통 시
- **축산물판매업 신고** — 동물성 유지(치유) 유통 시

동물성 유지(치유)는 **축산물가공업 허가 보유 OEM 공장 매칭 필수**.
- 일반 식품제조가공업으로는 합법적 양산 불가
- 진입장벽 ↑ (식물성 향미유 OEM 대비 MOQ·단가 부담)
- 한국 내 닭기름 향미유 전용 OEM 라인 적음 → 사전 컨택 필요

### 즉판 활용 가능 범위
아재꼬막의 즉석판매제조가공업으로 가능한 것:
- 매장 내 한정 제조·판매
- **온라인 통신판매 가능** (자사몰, 스마트스토어, 쿠팡, 마켓컬리 등 자기 셀러 계정)
- 9개월에 1회 자가품질검사
- 표시사항 의무 (제품명·원재료·유통기한·알레르기 등)

불가능한 것:
- 제3자 유통업자 도매 납품
- 마트·편의점 입점
- B2B 도매 거래

**R&D 시제품 자체 제조에 활용**:
- 향미유 R&D 의뢰 케이스에서 시제품 검증 채널로 활용 가능
- 매장 판매 데이터를 OEM 협상력 ↑ 레퍼런스로 활용
- 단, 동물성 유지(치유)는 즉판으로도 회색지대 — 지자체 확인 필요
