# 오재석 — 늘프레시 (BNI 멤버, 의뢰 추천자)

박준호의 BNI 조찬모임 멤버이자, 향미유 R&D 의뢰 케이스에서 의뢰처(늘프레시)와 박준호를 **이어준 추천·중개자**.

## 위치

- **소속**: 늘프레시 (프랜차이즈 유통사)
- **직위**: 팀장/이사급 (대표이사 채상훈의 가족관계)
- **접점**: 박준호의 BNI 조찬모임 — [[위키/주제/BNI_조찬모임_네트워킹]]

## 박준호와의 신뢰 형성

박준호가 BNI 모임에서 두 가지를 직접 제공:
1. **조찬 케이터링** — BNI 모임용 조찬을 박준호가 직접 준비
2. **데리야끼 소스 시식** — 박준호 R&D [[위키/주제/소스_RnD_진데리야끼]] v3.3 시식 제공

→ 오재석 대표가 박준호 음식을 **직접 먹고** 음식 신뢰 형성

## 의뢰 흐름에서의 역할

```
박준호 ── [BNI 멤버 + 음식 신뢰] ── 오재석 대표
                                       ↓ [추천·중개]
                                  [[위키/사람들/채상훈_늘프레시]]
                                       ↓ [최종 의사결정]
                                  향미유 R&D 의뢰 ([[위키/사업/향미유_RnD_의뢰_2026Q2]])
```
