# AZAE 디자인 시스템

박준호가 자기 사업·자기 자신을 외부에 시각적으로 표현하는 시스템. 단순한 색·폰트가 아니라 **박준호의 미감과 한식 본질**을 담은 시각 언어.

---

## 1. 컬러 팔레트 (확정)

| 역할 | 헥스 | 이름 |
|---|---|---|
| Primary | **#8B6F47** | 누룽지 브라운 |
| Accent | **#F5E8D3** | 갓지은 쌀밥 크림 |
| Background | **#FFFBF2** | 따뜻한 화이트 |
| Sub | **#E8D5B8** | 베이지 |

### 컨셉
**한식의 본질 (누룽지 + 쌀밥) + 모던 미니멀**

### 색 이름의 의미
박준호가 색 자체를 한국어로 명명. 헥스 코드보다 **이름이 먼저**. 즉, 박준호의 색은 단순 데이터가 아니라 **이미지·기억·감각**이 먼저.

| 이름 | 환기되는 것 |
|---|---|
| 누룽지 브라운 | 어릴 적 솥밥의 누렇게 탄 바닥, 한국 식문화 가장 깊은 베이스 |
| 갓지은 쌀밥 크림 | 막 지은 흰쌀밥의 부드러운 크림 톤, 일상의 따뜻함 |
| 따뜻한 화이트 | 차가운 회색이 없는 화이트, 인간적 톤 |
| 베이지 | 위 셋을 잇는 중간 톤 |

→ 박준호의 자기 진술 "**좋아하는 색깔: 누룽지 브라운 · 갓지은 쌀밥 크림**" 이 직접 사업 컬러로. **자기 미감 = 사업 디자인**.

## 2. 타이포그래피

| 용도 | 추천 폰트 | 시스템 fallback |
|---|---|---|
| 영문 헤드라인 | Cormorant Garamond / Playfair Display | Georgia |
| 한글 본문 | Pretendard / 본명조 | 맑은 고딕 |
| 코드/모노 | JetBrains Mono | Consolas |

### 페어링 원칙
**영문 세리프 1 + 한글 산세리프 1 조합**. 강의 권장 (최대 2개).

### 박준호식 해석
- **세리프 영문** = 격조·전통·신뢰.
- **산세리프 한글** = 현대성·가독성·일상.
- 둘의 결합 = **"전통의 무게 + 현대의 명료함"**. 박준호의 사업 컨셉 (한식 본질 + AI 시스템) 의 시각 번역.

## 3. 로고

### 현재 버전
- 일러스트 직접 제작 (AZAE 영문 세리프).
- 두 가지 변형: 갈색 배경 (#8B6F47) + 크림 배경 (#FFFBF2).
- 태그라인 (소형): "**손이 기억하는 맛**" — 현재 임시, 변경 검토 중.

### 로고의 의미
- AZAE 영문 = **"아재" 의 영문 표기**. 친근함을 격조로 전환한 작명.
- 세리프 = 전통주·고급 식품 브랜드 톤. 매타리(@matari.sool) 와 같은 결.
- 두 배경 변형 = 사용 맥락에 따른 페어 시스템.

## 4. 시그니처 시각 자산

박준호가 정의한 4개 핵심 시각:

1. **꼬막 양념 클로즈업** — 붉은 양념 + 검은 껍질의 컬러 충격. 시그니처 메뉴의 정체성.
2. **손으로 무치는 장면** — "**손맛 · 진정성**". 박준호의 프로 요리사 페르소나 ([[위키/생각/멀티페르소나_운영원칙]]) 가 시각화.
3. **갓 지은 쌀밥 + 꼬막 한 그릇** — 한식의 본질을 한 컷에. 컬러 팔레트의 직접 발현 (누룽지 + 쌀밥).
4. **사장(박준호) 얼굴** — 신뢰·실명 노출. Building in Public ([[위키/주제/SNS_콘텐츠전략_BuildingInPublic]]) 의 인간적 베이스.

→ 4개 시각 자산이 합쳐져 박준호 사업의 비주얼 정체성 형성.

## 5. 디자인 원칙 (박준호 자기 정의)

| # | 원칙 | 함의 |
|---|---|---|
| 1 | **차가운 회색·흰색 톤 금지** | 박준호의 따뜻한 톤과 안 맞음 |
| 2 | **세리프 영문 + 한글 명조 페어링** | 격조 + 가독성 |
| 3 | **여백 우선** | 캔바·인스타 모두 적용 |
| 4 | **꼬막의 컬러는 그대로 살리되, 디자인 톤은 누룽지·쌀밥** | 음식 자체와 브랜드 톤 분리 |

### 원칙 4의 정밀함
- 음식 (꼬막) 의 본래 색 = 붉은 양념 + 검은 껍질 (강한 콘트라스트).
- 브랜드 톤 = 누룽지 + 쌀밥 (부드러운 따뜻함).
- 즉, **음식은 음식대로 강하게**, **브랜드는 브랜드대로 부드럽게**. 두 톤이 충돌하지 않고 보완.

→ 박준호의 [[위키/생각/멀티페르소나_운영원칙]] 의 페르소나 분리 사고가 디자인에도 적용. **음식 페르소나 ≠ 브랜드 페르소나**, 둘 다 박준호의 일부지만 분리해 다룸.

## 6. 박준호의 미감과의 정합

### 6.1 자기 진술
박준호 자기 진술:
> "좋아하는 색깔: **누룽지 브라운 · 갓지은 쌀밥 크림**."

→ 박준호의 개인 미감이 사업 컬러와 100% 일치. 사업이 자기 자신의 외부화.

### 6.2 정서 자아 ([[위키/자아/자기상_자기묘사]]) 와의 정합
박준호의 정서 자아 = "마음 여리고 정 많음". 차가운 회색 톤이 어울리지 않는 이유가 여기에 있음. 디자인 시스템이 정서 자아의 외부 표현.

### 6.3 광주 / 한식 연결
박준호의 정신적 고향 = 광주 ([[위키/장소/광주_정신적고향]]).
누룽지·쌀밥은 한국 식문화의 가장 깊은 베이스. 광주 식문화도 그 베이스 위.
→ 디자인 시스템이 박준호의 지역 정체성과도 연결.

박준호의 벤치마크 ([[위키/주제/벤치마크_호랑이굴_마타리]]) 인 매타리(@matari.sool) 의 컬러 비중:

| 비중 | 매타리 컬러 |
|---|---|
| 30% | 블랙 |
| 30% | 아이보리/오프화이트 |
| 20% | 베이지/누룽지 |
| 10% | 금빛 술 |
| 10% | 자연 보색 |

박준호 자기 분석:
> "AZAE 팔레트(#8B6F47 + #F5E8D3 + #FFFBF2 + #E8D5B8)와 거의 동일한 결. **본인 선택이 옳았다는 직접 증거**."

→ 박준호가 자기 미감을 외부 사례와 교차 검증해서 자기 확신을 채움. ENTJ Crusader 의 [[위키/자아/MBTI_ENTJ_Crusader]] Type Angst 의 자기 의심을 데이터로 무력화.

## 10. 위험 / 주의

### 10.1 따뜻한 톤의 일관성 유지
- 누룽지·쌀밥 톤은 가게·SNS·로고 모두에 일관되게 적용해야 효과.
- 한 채널이라도 차가운 톤이 섞이면 시스템 무너짐.

### 10.2 슬로건 미결정
- 박준호 본인이 "손이 기억하는 맛" 을 임시로 둔 상태.
- AI가 임시 생성한 안. 박준호가 결정 대기.
- → 추후 인터뷰에서 박준호의 슬로건 결정을 추적.

### 10.3 시각 자산 촬영 미진행
- 디자인 원칙·컨셉은 정해져 있으나 실제 시각 자산 (꼬막 클로즈업·손 무침·쌀밥 한 그릇·사장 얼굴) 의 촬영이 미진행.
- SNS 시작 시점의 직접 병목.
