# 나는 누구인가 — 자아 영역 MOC

자아 폴더의 **지도**. 박준호의 정체성을 한 페이지로 압축하고, 깊이 들어가고 싶을 때 어느 노트로 가야 하는지 안내한다.

이 노트의 본문은 **새로운 사실을 만들지 않는다** — 모든 진술은 다른 노트에 출처가 있다.

---

## 0. 한 줄 정체성

> 박준호. 1992년생, 서울 용산 거주, 한식 가게 "아재꼬막"을 운영하면서 양념장 B2B·AI 에이전트·SNS 콘텐츠 등 다층 사업을 굴리는 사업가. ENTJ Crusader / INFP Ranger Paradoxitype. 자기 자신을 시스템으로 다루며, 자영업과 다음 시대 사이의 **징검다리** 가 되려 한다.

→ 출처: [[프로필_기본정보]] / [[MBTI_ENTJ_Crusader]] / [[변신로봇_시스템빌더_징검다리]]

---

## 1. 두 차원의 자기 정의

박준호의 자기상은 **두 차원**으로 갈라져 보존된다.

### 차원 A — 감정·태도 ([[자기상_자기묘사]])
박준호 자기 진술 한 단락 → 다섯 자아층:

| 층 | 내용 |
|---|---|
| 사회적 페르소나 | 요리덕후 + 그냥 일반인 |
| 추진 자아 | 도전 |
| 그림자 자아 | 어쩔 땐 나태하며 나약 |
| 정서 자아 | 마음 여리고 정 많음 |
| 일탈 자아 | 또라이끼 |

### 차원 B — 기능·역할 ([[변신로봇_시스템빌더_징검다리]])
학교 과제 자기 진술 → 세 키워드:

| 키워드 | 차원 | 답하는 질문 |
|---|---|---|
| **변신로봇** | 행동 방식 | 어떻게 일하는가 |
| **시스템 빌더** | 사고 방식 | 어떻게 생각하는가 |
| **징검다리** | 사회적 역할 | 사회에서 무엇을 하고 싶은가 |

→ 두 차원이 합쳐져야 박준호의 전체 모습. **감정의 여린 핵 + 시스템의 단단한 외피**가 동시에 작동.

## 2. 인지 모델 — ENTJ Crusader + INFP Ranger Paradoxitype

[[MBTI_ENTJ_Crusader]] 기준:

- **표면**: ENTJ — Te(즉결단) → Ni(원리) → Se(관찰) → Fi(의미)의 인지 순서. *Crusader* 특화 — "긍정적 변화를 만들기 위해 무엇이든 한다, 같은 일을 하도록 다른 사람을 이끈다."
- **숨겨진 핵**: INFP Ranger — 조용하고 사색적인 철학자. 자기만의 정신적 공간을 가꿈. 여린 마음·정·내면적 의미부여.
- **약점**: Fi (감정·의미의 디테일). 가장 늦게 점검됨.
- **함정**: Type Angst — "아무도 내 말 안 듣는다" → Trixie Tantrum (더 크게/빠르게 떠들기).
- **건강한 모습**: ENTJ 추진력으로 외부에 나서되, INFP Ranger의 의미·정·여린 마음을 숨기지 않을 때.

→ 위키 자체가 Ranger의 행위(자기만의 의미 공간 짓기)이며, ENTJ 시스템 사고로 운영되고 있음.

## 3. 자기 운영 시스템 — 다층 메타인지

박준호 안에는 의식적으로 분리·운영되는 **여러 층의 자기 시스템**이 동시에 돌아간다. 핵심 선언: *"나는 나를 믿지 않는다"* ([[../생각/나를_믿지않는다_다중메타인지]]).

| 층 | 도구 | 페이지 |
|---|---|---|
| 행동 트리거 (매일의 자기 대화) | 확언·자책·즉시행동·위기재해석 | [[../생각/자기에게_하는_말]] |
| 메타인지 절차 (사고 분리) | 런닝·걷기·자기 객체화 5단계 | [[../생각/안풀릴때_대처법_메타인지]] |
| 외부 시스템 (시스템 빌더) | 위키·AI 에이전트·다중 메타인지 | [[../생각/나를_믿지않는다_다중메타인지]] |
| 페르소나 운영 (변신로봇) | 5개 페르소나 의식적 전환 | [[../생각/멀티페르소나_운영원칙]] |
| 자기 인식 성향 (시스템 설계자) | 객체화·인식론적 신중함·시스템 사고 | [[메타인지_자기이해성향]] |

→ 박준호의 "자기 자신을 다루는 운영 체제(OS)". 의식적으로 설계 중.

## 7. 운영 메모

- 이 MOC는 **자아 폴더에 새 노트가 추가되거나 핵심 자기 진술이 변할 때** 갱신.
- 이 노트의 본문은 **요약·인덱스만** — 새로운 사실은 추가하지 않음. 사실은 항상 출처 노트에 있다.
- 출처가 모호하거나 추론이 들어간 진술은 위 본문에 *없도록* 유지.
