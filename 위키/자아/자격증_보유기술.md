# 자격증 / 보유 기술

박준호가 현재 보유하거나 응시 중인 자격증·기술 목록. 자격증 라인업은 **그 사람의 진로 의도를 가장 직접적으로 보여주는 지표** 중 하나다.

---

## 1. 보유·응시 자격 (확정)

| 자격 | 상태 | 비고 |
|---|---|---|
| **기중기 운전 자격증** | 보유 | 산업현장 자격 |
| **운전면허 1종 보통** | 보유 | 자가용·승합·소형화물 운전 가능 |

## 2. 자격증의 두 트랙

박준호의 자격증은 **시기적·성격적으로 두 묶음**으로 갈린다.

### 트랙 A — 산업·운전 트랙 (예전)
- 기중기 자격증
- 1종 보통

이 트랙의 의미:
- 토목공학과 → 전기공학과 시기와의 연관 가능성. 공학·산업 현장의 도구로 시작.
- "**또라이끼**" 의 한 단면 — 평범한 운전면허 외에 **기중기**라는 선택은 흔치 않음. 일반인에게서 보기 드문 카드.
- 1종 보통은 차량을 직접 운전하고 짐을 실어 나를 수 있다는 의미 = **자기 발로 움직이는 인프라** 보유.

### 트랙 B — 식음·외식 트랙 (현재)
- 식품가공기능사 (필기 합격)
- **식품가공기술사** (2026-05-14 응시 계획 확정)

이 트랙의 의미:
- **호텔외식경영학과 + 요리덕후 + 식품가공기능사 + 식품과학 학습 + 식품가공기술사** = 진로 트랙이 식음으로 명확히 정렬 + **과학·기술 깊이까지 확장**.
- 박준호 본인 표현 ([[원본/대화/2026-05-14_정체성-정정-늘프레시]]): "과학을 잘 하고 좋아해서 식품과학공부도 하고 있어서 식품가공기능사 자격증도 현재 필기는 합격한 상태 기술사까지 따기위해 공부를 할 예정이야".
- 기능사 → 기술사로 **상위 자격 추구** = ENTJ Crusader의 "사다리 한 칸 위로" 사고가 자격 영역에서도 작동.
- 식품가공기술사는 박준호의 [[위키/사업/azae_컨설팅_사업모델]] (R&D 컨설팅)·[[위키/사업/멜트컴퍼니_법인_운영]] (식품 제조 법인) 의 **공신력 자산**으로 작동할 수 있음.
- → ENTJ Te(즉결단·실행)의 지금-여기 작동 사례.

## 3. 다음에 추가될 가능성 (추정)

식품가공기능사 실기 합격 후, 박준호의 진로 방향을 따라가면 다음 자격증이 자연스럽게 후속될 가능성이 있다 (어디까지나 추정 — 본인 의지 확인 필요):

- **조리기능사 (한식·양식·중식·일식 중)** — 요리덕후 + 호텔외식 정렬.
- **위생사** — 외식 운영의 위생 관리.
- **식품기사** — 식품가공기능사의 상위 자격.
- **소믈리에 / 바리스타 / 제과·제빵기능사** — 외식의 인접 영역.
- **사업자등록 / 식품 영업 신고** — 창업 트랙의 필수 단계.

→ 박준호의 다음 인터뷰에서 다음 자격 계획을 확인하면 진로의 방향성이 더 명확해짐.
