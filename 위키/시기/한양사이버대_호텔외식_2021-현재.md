# 한양사이버대 호텔외식경영학과 (2021~현재)

박준호의 인생에서 **세 번째 대학 시도이자, 진로 트랙을 공학에서 음식으로 완전히 바꾼 시기**. 현재진행형이며 곧 졸업을 앞두고 있다.

## 2. 진로 전환의 의미 — "공학 → 음식"

### 이전 트랙 (자퇴)
- 전남대 토목공학과 (1차)
- 광운대 전기공학과 (2차)

### 현재 트랙
- 한양사이버대 **호텔외식경영학과**

### 분석
- **완전한 트랙 변경**: 공학(이공계 → 단단한 물리·전기) → 외식(서비스·음식·운영). 거의 정반대 영역.
- 박준호의 자기 묘사 "**요리덕후**"와 정확히 일치하는 학과 선택.
- 2015년 광운대 자퇴 → 2021년 한양사이버대 입학 사이의 **6년 공백기**를 거쳐 도달한 결정 = 충분히 숙성된 선택일 가능성 큼.
- → ENTJ N(Ni)의 작동: 시간을 두고 "내가 진짜 가야 할 길은 이쪽이다"의 원리(Ni)를 정리하고, Te(즉결단)으로 입학을 실행.

### 사이버대 선택의 의미 — 2차 인터뷰로 부분 확인
박준호 자기 진술:
> "호텔외식경영은 **석사를 위해 관련 학과를 지원해서 다닌 거**라 큰 생각이 없어."

→ 즉, 사이버대 선택의 핵심 동기는 **경희대 대학원 진학에 필요한 학사 자격을 효율적으로 채우기** 위한 것 ([[위키/질문/경희대_대학원_진학]]).

또한 박준호는 학사 재학 기간 내내 **아재꼬막 가게를 운영**하고 있었으므로:
- 사이버대 선택 = **가게 운영을 멈추지 않고 학력 자격을 채우는** 가장 합리적 경로.
- 일·생계와 병행 가능, 자기 페이스 운영, 비용 효율 — 모두 박준호의 현실에 맞는 정합적 선택.
- → ENTJ Te(효율적 단계 실행) 의 정공법.

## 3. 자격증 라인업과의 정렬

박준호는 **식품가공기능사 필기에 합격했고 실기를 앞두고 있음**. 이는 호텔외식경영학과의 학업과 별개로 진행하는 **현장 자격증** 트랙.

→ 학과(경영·이론) + 자격증(현장·기능) + 자기 정체성(요리덕후)의 **3단 결합** = 단순한 학위 취득이 아니라 진짜 진로로 향하는 행보.
→ 페이지: [[위키/자아/자격증_보유기술]] / [[위키/주제/요리_음식_커리어방향]]

## 4. 현재 시점의 위치

2026년 5월 = **마지막 학기 진행 중**. 의미:

1. **곧 졸업 = 새로운 트랙의 첫 학위**. 12년간(2011 첫 입학~2026 졸업) 학력의 마침표.
2. **다음 단계 결정의 시기** — 취업? 창업? 추가 학습? 어떤 방향인지가 가장 중요한 시점.
3. **인생의 매듭 짓는 학기** — 두 번의 자퇴를 거쳐 도달한 졸업의 무게.

→ **위키를 시작한 타이밍이 의미심장함**. 자기 자신을 정리하기 시작한 것이 바로 "졸업 직전, 다음 단계 결정 직전"이라는 것은 우연이 아닐 가능성. 다음 진로를 정확히 선택하기 위해 자기 데이터를 모으고 있다는 해석 가능.
