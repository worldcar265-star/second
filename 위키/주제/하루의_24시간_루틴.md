# 하루의 24시간 루틴

박준호가 매일 어떻게 살고 있는지의 구체적 시간 흐름. 이 페이지는 다른 모든 활동(가게·AI 빌딩·운동·후추·콘텐츠) 이 24시간 안에서 어떻게 박혀 굴러가는지를 한 자리에 정리.

---

## 1. 영업일 — 시간 흐름 (자기 진술 정리)

| 시간대 | 활동 | 페르소나 |
|---|---|---|
| **6~7시** | 기상 + 간단한 운동 | (시동) |
| **오전** | 클로드 공부 + 커피 + 위키 작성 + 에이전트 구축·자동화 고민 | 시스템 빌더 |
| **11~12시** | **가게 출근** | 사장 / 요리사로 전환 |
| 출근 후 | 식사 → 재료 준비 → 영업 시작 | 요리사 + 사장+서버 |
| 영업 시간 | 가게 운영 (몸의 일) | 요리사 + 사장+서버 |
| **마감 후** | **12시 반 ~ 1시 집 도착** | (모드 종료) |
| 집 도착 후 | 후추 발라당 → 만져주기 → 같이 잠 | (휴식) |
| 1시경 | **취침** | (수면) |

→ 박준호의 영업일은 **오전(머리) + 오후(몸) + 새벽(취침)** 의 3 단 구조. 이 사이에서 **수면 5시간** 만 확보.

---

## 2. 휴무일 — 월요일 1일

박준호 자기 진술:
> "휴무일은 지금은 월요일만 쉬고 있어."

### 휴무일 시간 흐름
| 시간대 | 활동 |
|---|---|
| **6~7시** | 기상 (영업일과 같음) |
| 아침 | SNS 많이 봄 |
| 오전 | 약속 없으면 클로드 작업 (영업일 오전과 동일) |
| 낮 | 미팅 / 사람 만나기 / 평일 못한 일 / 미뤄둔 일 |
| 평일 | (집안일 필요한 날엔 오전에) |

### 휴무일의 의미
- 기상 시간이 영업일과 같음 → 박준호의 신체 리듬이 **요일 단위로 흔들리지 않음**.
- 휴무일 ≠ 쉬는 날. 휴무일 = **사람·미팅·미뤄둔 일을 처리하는 날**.
- 박준호의 "**쉬는 게 무엇인지 잘 모르는 사람**" 자기 진술 ([[위키/생각/쉬는것이_어렵다]]) 의 직접 증거.

## 3. 에너지 곡선

박준호 자기 진술:
> "**밥을 먹기 전 오전 시간이 가장 쌩쌩**한 거 같아. **10~11시부터는 거의 죽어 있지**. 그래서 오전 중에 머리를 쓰는 일을 가장 많이 해."

### 박준호의 에너지 분포
```
[06시] ████████████ 최고 (기상 + 운동 + 커피)
[09시] ██████████░░ 높음
[11시] █████░░░░░░░ 급락
[12시] ███░░░░░░░░░ 죽어 있음 (밥 + 가게 일)
[20시] ██░░░░░░░░░░ 낮음
[24시] █░░░░░░░░░░░ 마감 후 잠
```

### 함의
- **머리의 일(AI 빌딩·위키·신메뉴 R&D·전략) 은 오전에 박혀 있어야** 한다.
- 오후 = 몸의 일 (가게)에 적합.
- 박준호의 시간 압박은 **머리의 일에 쓸 수 있는 절대 시간이 오전 4~5시간뿐** 이라는 사실에서 비롯.

→ 페이지 [[위키/질문/매장운영과_비즈니스_동시병행]] 의 시간 부족이 이 곡선에서 정량화됨.

## 4. 잠 / 식사 / 운동 (정량)

### 4.1 수면
- **하루 약 5시간**.
- 자연 기상 (알람 없음 / 약함 — 추후 확인).
- 일반 권장(7~8시간) 보다 약 2~3시간 적음. **만성 수면 부족 상태**.

### 4.2 식사
- 하루 **1~2끼**.
- 가게에서 해결.
- 본인 표현: "**대충 먹어**". 컵라면 / 가게에 있는 걸로 간단히.
- → **요리덕후가 자기 끼니를 가장 못 챙기는** 역설. 박준호의 [[위키/생각/행복이_고장났다]] 의 "사소한 행복을 못 즐긴다" 와 결이 닿음.

### 4.3 운동 (정량 — 가장 정밀하게 진술된 영역)
박준호의 운동 루틴은 매우 구체적. 별도 페이지: [[위키/주제/운동_런닝과_홈트]]

요약:
- **주 2회 런닝** — 5km, 페이스 4'40"/km, 한강 코스 (걸어가서 3.5km 지점 회차).
- **런닝 안 하는 날** — 팔굽혀펴기 40 × 2 세트, 스쿼트 30 × 2 세트.

### 4.4 자극제
- **커피**: 하루 자주 (양 미확인).
- **흡연**: 안 함 (마지노선, [[위키/생각/윤리_마지노선과_타협의정의]]).
- **술**: 모임/자리 있을 때만 가끔. **모임 자체에 거의 안 가서 한 달 1번도 안 마심**.

→ 카페인 외에는 자극제 거의 없음. 박준호의 마지노선이 일상 음용에서 그대로 작동.

## 5. 머리 굴리는 시간 — 절대량과 분포

### 5.1 AI 에이전트 빌딩 / 위키 작업
박준호 자기 진술:
> "오전에 걸어다니며 혹은 집에서 가게 오픈 전까지 하고 있어. 휴무일 한 덩어리로 할 때도 있고, 조각 시간에도 할 수 있고. **끊임없이 하고 있다고 봐도 무방해.**"

→ 박준호의 머리 시간은 **오전 한 덩어리 + 휴무일 한 덩어리 + 모든 조각 시간**. 즉, 깨어 있는 거의 모든 비매장 시간이 머리 일.

### 5.2 현재의 집중 영역
박준호 자기 진술:
> "요즘엔 AI 공부를 **위키에만 집중**하고 있어. **Git이랑 커밋** 그런 기본적인 것들을 익히고, **베이스를 최대한 잘 잡아놓고 가야 그 뒤에 안정적인 에이전트 구축이 가능**하다고 생각하고 있거든."

→ 박준호의 [[위키/주제/AI에이전트_개발]] 가 현재 위키 베이스 구축 단계에 있음. 이건 ENTJ 시스템 빌더의 정공법 — **인프라부터 단단히**.

### 5.3 콘텐츠 / 카메라 앞 페르소나 — 미진행
박준호 자기 진술:
> "콘텐츠는 지금은 **하는 일이 너무 많아서 아직 진행을 안 하고 있어**. 사실 법인 빌드 하는 걸 찍어야 한다고 생각은 하는데, 지금 이거까지 생각하면서 **페르소나 갈아끼우고 편집하고 하기에 솔직히 버거워**."

→ [[위키/주제/SNS_콘텐츠전략_BuildingInPublic]] 가 정해져 있지만 **시간 압박으로 미진행**. 5 페르소나 중 "감독" 페르소나의 가동 시점이 미뤄짐.
→ [[위키/질문/매장운영과_비즈니스_동시병행]] 페이지의 시간 부족 결과가 이 영역에서 직접 드러남.

## 6. 후추와의 일상 — 짧지만 정해진 의식

박준호 자기 진술:
> "후추와는 많은 시간을 보내지 못해ㅠㅠ. 집에 오면 발라당 하면 만져주고 잠 같이 자고, 아침에 일어나면 **고로롱 타임**에 만져주고 그게 거의 다야. 아침에 컴퓨터 하고 있으면 주변에 있어서 괴롭히거나 놀거나 하긴 하고 있지 요즘엔."

### 박준호와 후추의 정해진 의식
| 시간 | 의식 |
|---|---|
| 새벽 1시 (귀가) | 후추 발라당 → 박준호 만져줌 → 같이 잠 |
| 아침 (기상 후) | 고로롱 타임 → 박준호 만져줌 |
| 오전 (컴퓨터) | 후추가 주변 / 괴롭히기·놀이 |

→ 시간은 짧지만 **하루 두 번의 정해진 접촉 + 오전 동안의 곁** 이 박준호와 후추의 관계 패턴.
→ "ㅠㅠ" 는 박준호 자기 진술에서 흔치 않은 감정 표현. **시간을 못 내주는 것에 대한 미안함**의 직접 표시.

→ [[위키/사람들/후추_고양이]] 갱신.

## 7. 집 — 박준호의 정의

박준호 자기 진술:
> "**집은 집에서 재충전이면서 아이디어를 구상하는 사무실과 같아.**"

### 박준호의 집의 두 역할
1. **재충전 공간** — 잠 + 후추와의 짧은 시간.
2. **아이디어 구상 사무실** — 오전 머리 일의 무대.

→ 박준호의 집 ≠ 휴식만의 공간. 집 = **저강도 사무실 + 침실**. 가게가 고강도 사무실(몸) 이라면, 집은 저강도 사무실(머리).
→ [[위키/장소/용산구_한강대로_현재거주지]] 갱신.

## 8. 사람과의 시간 — 박준호의 자기 진단

박준호 자기 진술:
> "지금은 혼자 거의 많은 시간들을 보내고 있고, 사람과 함께하는 시간은 일하는 시간을 제외하고는 휴일에는 거의 혼자 있는 듯해.
> 근데 혼자 시간에 기울어져 있지만 **난 사람과 함께하는 시간이 좀 더 필요한 거 같아.**
> **반반 정도였으면 좋겠어.**"

→ 박준호 본인의 명확한 자기 진단:
- **현재 비율** = 혼자 우세 (휴일도 거의 혼자).
- **이상 비율** = 50:50.
- **조정 의지 있음**.

→ 별도 페이지로 추적: [[위키/질문/사람과의_시간_부족함]].

## 9. 24시간 안의 5 페르소나 분포

박준호의 [[위키/생각/멀티페르소나_운영원칙]] 5 페르소나가 24시간 안에서 어떻게 작동하는지:

| 페르소나 | 영업일 시간 | 휴무일 시간 |
|---|---|---|
| 시스템 빌더 | 오전 4~5시간 | 약속 없으면 거의 풀타임 |
| 사업가 (책상) | 오전 안에 포함 / 미팅 시 | 미팅 / 휴무일 미뤄둔 일 |
| 프로 요리사 (주방) | 11~12시~마감 안에 | (휴무) |
| 사장 + 서버 (홀) | 영업 시간 안에 | (휴무) |
| **감독 (카메라 앞)** | **현재 미가동** | **현재 미가동** |

→ 박준호의 5 페르소나 중 **감독 페르소나만 현재 가동되지 않음**. 이건 SNS·법인 빌드 콘텐츠가 미뤄지는 직접적 결과.

## 10. 일주일 단위 패턴

| 요일 | 활동 |
|---|---|
| 화요일 | 영업 + **BNI 조찬모임** ([[위키/주제/BNI_조찬모임_네트워킹]]) |
| 수요일 | 영업 |
| 목요일 | 영업 |
| 금요일 | 영업 |
| 토요일 | 영업 |
| 일요일 | 영업 |
| **월요일** | **휴무** |

→ 일주일에 **6일 영업 + 1일 휴무 + 1일은 매주 BNI 화요일 새벽 일정**. 박준호의 주간 리듬이 **6.5일 가동 + 0.5일 휴무**.

## 11. 위험 / 주의

### 11.1 수면 5시간의 누적 부담
- 만성 수면 부족이 ENTJ Type Angst (자기 의심) 와 [[위키/생각/행복이_고장났다]] 의 정서 처리에 악영향 가능.
- 가장 우선적으로 점검할 항목.

### 11.2 식사의 부실
- 요리덕후가 자기 끼니를 컵라면으로 때우는 패턴.
- 단기 운영의 효율 우선 + 자기 돌봄 후순위.

### 11.3 감독 페르소나 부재의 누적 비용
- SNS 미시작 = B2B 리드 미발생.
- 법인 빌드 콘텐츠 미촬영 = 자산화되지 않은 시간.
- 박준호 본인이 "**버겁다**" 진술 — 시스템에 페르소나 한 개 더 끼워넣을 여유가 없음. AI 에이전트가 이 자리를 메울 수 있는지 위키가 추적.

### 11.4 사람과의 시간 부족 — 본인 인지
- 박준호 본인이 명시적으로 부족 감지.
- 9인 생일 모임·고향친구·BNI 조찬모임이 현재 사람 채널의 거의 전부.
- 의도된 보강이 위키 추적 항목.
