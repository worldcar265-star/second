# BNI 조찬모임 — 박준호의 주간 네트워킹 채널

박준호가 매주 화요일 새벽 참여하는 비즈니스 네트워킹 모임. 단순 친목이 아니라 **AZAE 사업의 주요 외부 채널**로 작동. 4차 인터뷰에서 처음 등장한 새 정보.

---

## 1. 박준호 자기 진술 (원문)

> 매주 화요일 나가는 **BNI 조찬모임**에서 저번 주에 **소스 샘플을 들고 가서 위클리 발표**를 했더니, **투자 쪽 사람과 정책자금 관련 업계 사람들이 대화하자고 하는 게 굉장히 흥미로웠고**.
> **그게 미팅으로 이어져서 투자와 법인 전환에 대해 이야기 나누는 것도 무척이나 즐거웠어.**

---

## 2. BNI 일반 정보

BNI (Business Network International) = 1985년 미국에서 시작된 비즈니스 레퍼럴 네트워킹 단체. 한국에도 다수 챕터 운영.

### BNI 의 특징
- 주 1회 정기 조찬 모임 (보통 화요일 또는 목요일 새벽).
- 한 챕터당 같은 업종 1명 (배타적 네트워킹).
- 위클리 발표 (1분 자기 사업 소개) 의무.
- 멤버 간 **레퍼럴 (잠재 고객 추천) 교환**.
- 박준호의 챕터·지역·가입 시점은 미진술.

## 3. 박준호의 BNI 활용 — AZAE 사업 채널

### 3.1 박준호의 모임 위치
| 항목 | 추정 / 확인 |
|---|---|
| 가입 챕터 | 미확인 |
| 박준호의 업종 슬롯 | 외식업 / 식품 제조 (AZAE) — 추정 |
| 가입 시점 | 미확인 |
| 참석 빈도 | **매주 화요일** (정기 참석) |

### 3.2 핵심 사건 — 소스 샘플 발표 (지난주)
박준호의 자기 진술에서 가장 중요한 사실:

> "**소스 샘플을 들고 가서 위클리 발표**" → "투자 쪽 + 정책자금 업계 사람들 대화 요청" → "**미팅으로 이어짐**" → "**투자와 법인 전환 논의**"

이건 단순 모임 후기가 아니라 **AZAE 사업의 핵심 마일스톤**:

```
[AZAE 양념장 개발] → [BNI 위클리 발표 (샘플 시연)] → [관심 업계 사람 발생]
                                                      ↓
                              [미팅 성사] → [투자 + 법인 전환 논의]
```

→ 박준호의 [[위키/주제/양념장_B2B_사업]] 의 "**투자 유치 + 법인 전환**" 이 추상적 계획이 아니라 **이미 진행 중**임을 보여주는 첫 외부 검증 사건.

---

## 3.3 핵심 사건 — 케이터링·시식 → 음식 신뢰 → 의뢰 수주 (2026-05-14 신설)

박준호가 BNI 활용을 한 단계 더 끌어올린 사례. **소스 샘플 발표(3.2)** 보다 더 깊은 신뢰 형성 방식:

### 박준호의 두 가지 직접 제공
1. **조찬 케이터링** — 박준호가 BNI 모임용 조찬을 직접 만들어 제공
2. **데리야끼 v3.3 시식** — 박준호 R&D 자산 ([[위키/주제/소스_RnD_진데리야끼]]) 시식 제공

### 신뢰 형성 → 의뢰 수주
```
박준호 ──[케이터링 + 데리야끼 시식]── [[위키/사람들/오재석_늘프레시]]
                                              ↓ [음식 신뢰 형성]
                                              ↓ [향미유 R&D 의뢰 추천]
                                          [[위키/사람들/채상훈_늘프레시]]
                                              ↓ [최종 결정]
                                       [[위키/사업/향미유_RnD_의뢰_2026Q2]]
```

박준호 본인 표현:
> "오재석 대표님은 내가 한번 조찬모임에 조찬을 준비해서 케이터링 한 적 있는데 그것도 먹고 데리야끼 소스도 만든거 드린적이 있어서 그거 먹고 이제 음식에 대한 신뢰는 생겨있는 상태야."

### BNI 활용의 두 모드 비교

| 모드 | 사례 | 결과 |
|---|---|---|
| **샘플 발표** (3.2) | 양념장 샘플 들고 위클리 발표 | 투자·정책자금 업계 관심 → 미팅 |
| **케이터링·시식** (3.3) | 조찬 직접 준비 + 데리야끼 시식 | 음식 신뢰 → R&D 의뢰 추천·수주 |

→ **샘플 발표 = "이런 거 만든다" 어필**, **케이터링·시식 = "내 음식 직접 먹어봐" 신뢰 자산화**. 두 모드 모두 박준호의 시스템 빌더 + 프로 요리사 페르소나의 조합으로 작동.

## 4. BNI 가 박준호에게 주는 것 — 4 가치

### 4.1 외부 검증 채널
- 박준호의 사업 방향이 외부 업계 사람에게 어떻게 받아들여지는지 매주 확인.
- ENTJ Type Angst ("내 말 아무도 안 듣는다") 의 자연스러운 무력화 도구.

### 4.2 잠재 B2B 고객·파트너 발견
- 투자자 / 정책자금 업계 / 프랜차이즈 / 케이터링 등이 모일 가능성.
- 박준호의 [[위키/주제/AZAE_브랜드_사업체계]] B2B 3채널 (라인 3) 의 직접 영업 무대.

### 4.3 사람과의 시간
- 박준호의 4차 인터뷰 자기 진단:
  > "사람과 함께하는 시간이 좀 더 필요한 것 같아. 반반 정도였으면 좋겠어."
- BNI = 매주 정기적으로 보장된 **사람 시간**. 일주일에 한 번 강제로 사람 만나는 채널.
- → [[위키/질문/사람과의_시간_부족함]] 의 부분적 해법.

### 4.4 강제 발표 = 자기 정리 도구
- 위클리 1분 발표 의무 = 매주 자기 사업을 한 줄로 압축해야 함.
- 박준호의 위키·메타인지·AZAE 한 줄 정의가 모두 같은 결의 자기 외부화 작업.
- BNI 발표가 박준호의 시스템 빌더 정체성과 정합.

## 5. 박준호의 일주일 안에서의 BNI 위치

박준호의 [[위키/주제/하루의_24시간_루틴]] 의 주간 패턴:

| 요일 | 활동 |
|---|---|
| 월요일 | 휴무 |
| **화요일 새벽** | **BNI 조찬모임** |
| 화요일~일요일 영업일 | 영업 |

→ BNI 는 박준호의 일주일에서 **유일하게 새벽 시간에 배정된 외부 일정**. 박준호의 신체 리듬(6~7시 기상) 과 정합.

## 6. 사업 관점의 의미 — "**소스 샘플** 만으로 미팅이 발생했다"

### 박준호 사업 모델의 외부 검증
박준호의 사업 자세:
> "**나는 B2B를 받지만, 그들은 B2C가 쉬워지는 비즈니스**." ([[위키/생각/일_가치관_문제해결_자유]])

BNI 사건이 이걸 증명:
- 박준호 (B2B 받는 쪽) → 투자자·정책자금 업계 (B2B 고객) 가 관심.
- 즉, 박준호의 사업이 **외부 업계 시각에서 자본 투입 가치 있는 모델로 인식**됨.

### "**즐거웠어**" 라는 박준호의 감정 표현
박준호는 자기 감정을 자주 드러내지 않음. 이번 진술에서:
- "**굉장히 흥미로웠고**"
- "**무척이나 즐거웠어**"

→ 박준호가 명시적으로 즐거움을 표현한 드문 순간. **"사업이 진척이 생길 때도 좋아"** 라는 박준호의 다른 자기 진술과 정합.

→ 페이지 [[위키/생각/행복이_고장났다]] 의 "박준호가 행복을 느끼는 좁은 통로" 에 **"꿈꾸던 자기 모습대로 살고 있을 때"** 가 포함됨. BNI 발표 → 미팅 성사 = 그 순간의 한 사례.
