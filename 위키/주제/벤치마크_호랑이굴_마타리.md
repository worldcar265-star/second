# 박준호의 벤치마크 — 호랑이굴 / 마타리

박준호가 자기 사업·SNS 전략의 참조 사례로 명시한 두 브랜드. 무작위로 고른 게 아니라 **박준호의 사업 모델의 두 축** 을 각각 대표하는 사례.

---

## 1. 두 벤치마크의 역할 분담

| 벤치마크 | 박준호에게 주는 것 |
|---|---|
| **호랑이굴** | Building in Public 전략 + 멀티브랜드 모회사 구조 |
| **마타리 (MATARI)** | 한식 본질 + 모던 미니멀 + B2B 협업 콘텐츠화 |

→ 두 사례를 합치면 박준호의 [[위키/주제/AZAE_브랜드_사업체계]] 가 그대로 그려짐. 박준호의 벤치마크 선정 자체가 자기 사업 그림의 직접 표현.

## 2. 호랑이굴 (@horangigul)

### 2.1 기본 정보 (박준호 정리 시점)
| 항목 | 내용 |
|---|---|
| 업종 | 한우대창 전문점 |
| 위치 | 범계역 (안양) |
| 운영사 | 브레멘F&B (호랑이굴 / 파랑새야 / 사자갈비 멀티브랜드) |
| 팔로워 | 4,418명 |
| 게시물 | 719개 |
| 특이점 | 오픈 6개월 전부터 인스타 운영 / 초기 200명이 거의 다 실방문 / 연매출 50억 |

### 2.2 박준호 선정 이유
- **Building in Public** 전략의 한국 외식업 성공 사례.
- **단일 시그니처 (한우대창) 집중**.
- **멀티브랜드 모회사** 구조 = AZAE 가 미래에 가질 구조 (아재꼬막 + AZAE 제조 + 컨설팅을 한 우산).

### 2.3 박준호가 적용할 4 가지
박준호 자기 정리:
1. 법인 전환·에이전트화 과정을 매주 1회 게시.
2. 단일 시그니처 (꼬막) 집중.
3. 사장 얼굴 자주 등장 = 신뢰.
4. 멀티브랜드 모회사 구조 참고.

### 2.4 중요한 데이터 — "오픈 6개월 전부터 인스타"
- 오픈 시점에 이미 인스타 운영 중이었음.
- 초기 방문 200명이 **거의 다 인스타 팔로워의 실방문**.
- → SNS 가 단순 마케팅이 아니라 **실제 매출의 직접 채널** 로 작동한 사례.
- 박준호의 [[위키/주제/SNS_콘텐츠전략_BuildingInPublic]] 의 "**팔로워 100명도 다 잠재 B2B 고객**" 통찰의 외부 증거.

## 3. 마타리 / MATARI (@matari.sool) ⭐ — 박준호의 핵심 벤치마크

### 3.1 기본 정보 (박준호 정리 시점)
| 항목 | 내용 |
|---|---|
| 업종 | 한국 전통주 (맑은술) 복원 |
| 위치 | 서울 |
| 인증 | ✓ (Verified) |
| 팔로워 | 5,952명 |
| 게시물 | **73개** (양보다 질) |
| 팔로잉 | 6명 (의도된 비대칭) |
| 외부 링크 | 3개 |

### 3.2 박준호 선정 이유
> "**본인과 가장 가까운 벤치마크**" — 박준호 자기 진술

- 한식 본질 + 모던 미니멀 = 박준호의 디자인 시스템과 동일 결.
- B2B 협업 콘텐츠화 = AZAE 의 라인 3 (B2B 납품 3채널) 의 표현 모형.
- 양보다 질의 콘텐츠 운영 = 박준호의 "**밀도 우선**" 결정의 모범.

### 3.3 박준호가 추출한 핵심 인사이트 5가지

#### 인사이트 1 — 고정 핀 3개 = 브랜드 본질의 응축
- 매타리: 결과물 (금빛 술잔) → 원료 (흰 쌀) → 과정 (누룩 덩어리).
- AZAE 차용: **꼬막무침 (결과) → 꼬막 원물 (원료) → 양념 만드는 과정**.
- → [[위키/주제/SNS_콘텐츠전략_BuildingInPublic]] 의 고정 핀 구조의 직접 출처.

#### 인사이트 2 — 73개로 5,952 = 콘텐츠당 81명
- 양보다 질의 압도적 효율.
- 박준호의 "**릴스 소비, 팔로우 안 함**" 통찰의 가장 강력한 외부 증거.
- 인스타 알고리즘이 **한 편의 좋은 콘텐츠를 누적적으로 노출**한다는 사실을 데이터로 확증.

#### 인사이트 3 — 하이라이트 = 사업 구조 그대로
- 매타리: Order / Partners / Essence / With / Malgeun Sool.
- AZAE 차용: **Order / B2B / Essence / With / AI Lab**.
- → 하이라이트가 단순 분류가 아니라 **사업 라인의 5 폴더**로 작동.

#### 인사이트 4 — 자체 IP 시리즈 운영
매타리의 IP:
- MATARI PLAYLIST (음악 페어링).
- MATARI JOURNAL (저널).
- 결들임 (페어링 큐레이션).
- Soolsang 술상 (이벤트).
- Moments of Connection (감성 영상).

박준호 차용 후보:
- 「아재 저널」
- 「꼬막상」
- 「AI 일지」

→ **콘텐츠 IP 회사 모델**. 가게 계정이 아니라 미디어 회사처럼 운영.

#### 인사이트 5 — B2B 협업 = 콘텐츠 자산화
매타리의 협업 사례:
- MOSU by SUNGANN (미슐랭 다이닝).
- 한국의집 Korea House.
- 월삼청 WOL SAMCHEONG 이벤트.

→ 매타리는 모든 협업을 **이벤트 콘텐츠로 자산화**. AZAE 의 라인 3 (프랜차이즈·케이터링·역공급) 에도 같은 접근. 협업이 끝나도 콘텐츠는 남음.

## 4. 두 벤치마크의 차이 — 박준호의 결합

### 호랑이굴이 보여주는 것
- 외식업 + Building in Public 의 **매출 효과**.
- 단일 시그니처 + 멀티브랜드 모회사 구조.

### 마타리가 보여주는 것
- 한식 + 모던 미니멀 의 **브랜드 격조**.
- 콘텐츠 IP 회사로의 운영 방식.
- B2B 협업의 자산화.

### 박준호의 결합
- **호랑이굴의 매출 효과 + 마타리의 브랜드 격조 = AZAE**.
- 박준호의 사업이 두 벤치마크를 합친 자리에 있음을 자기 자신이 자각.

## 5. 위험 / 주의

### 5.1 벤치마크 모방의 함정
- 두 사례를 너무 직접 모방하면 박준호의 차별점 (AI 컨설팅·자영업 시대 진단) 이 묻힐 수 있음.
- 보호 장치: 박준호의 **콘텐츠 카테고리 비중** ([[위키/주제/SNS_콘텐츠전략_BuildingInPublic]]) — AI·시스템 콘텐츠 65%, 가게·요리 20%. 매타리·호랑이굴 어느 쪽도 가지지 못한 비중.

### 5.2 매출 규모의 차이
- 호랑이굴은 연매출 50억 의 멀티브랜드 회사. AZAE 는 1인 사장 + 3년차 가게.
- 호랑이굴의 결과를 그대로 기대하지 않고 **단계적으로** 따라가는 것이 박준호의 [[위키/생각/일_가치관_문제해결_자유]] 사다리 사고.

### 5.3 매타리 데이터의 시간성
- 매타리의 팔로워·게시물 수는 박준호 정리 시점(2026-05-03) 기준. 시간이 지나면 변함.
- 위키는 데이터를 박제하지 않고 **벤치마크의 운영 원리**를 추출하는 것이 목적.
