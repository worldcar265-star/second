# 아재꼬막 채용 기준 — 4단 필터와 통제 밖 리스크

박준호가 [[위키/주제/아재꼬막_가게]] 의 아르바이트생을 평가할 때 실제로 작동시키는 의사결정 알고리즘. 2026-05-05 채용(24살 남 vs 35살 여 → 35살 여 채용) 사후 분해를 통해 추출된, 박준호 자기 진술 기반의 우선순위 구조.

---

## 1. 4단 필터 (우선순위 순)

박준호는 "선호 항목을 나열하고 점수를 매기는" 방식이 아니라, **사다리식**으로 단계별 필터를 통과시킨다. 위 칸이 흔들리면 아래 칸은 무의미하다는 게 박준호 본인의 명시적 사고.

### 1순위 — 요리 가능
> "가게가 지금 중심이기때문에 가게가 흔들리면 아무것도 못하고 잇긴해."

- **가게(현재 중심) 안정화**가 모든 기준을 누른다.
- 박준호는 미래 능력(영상 등)이 아무리 매력적이어도, **현재 주방이 굴러가지 못하면 그 위에 쌓을 사다리 칸이 다 무너진다**고 본다.
- → [[위키/생각/일_가치관_문제해결_자유]] 의 사다리 구조의 직접 작동 사례.
- 자기 진술로 명시한 반사실(counterfactual): "정반대 스펙(24살 여 요리 가능 + 35살 남 영상 가능)이었다 해도 결정은 똑같았을 것" → 1순위가 성별·나이보다 위에 있음을 박준호 본인이 검증.

### 2순위 — 동년배
> "서른다섯살이어서뽑은것도있어같은나이고 스물네살은 다른기회나 다른선택지가열려있어서빨리그만둘수 생각한여지가있었거든."

- 같은 또래 = 소통 비용 낮음 + **이탈 리스크 낮음**.
- 어린 친구는 다른 기회가 열려있어 빨리 그만둘 가능성이 박준호 머릿속에서 가산점/감점으로 작동.
- 또한 "동년배여서 약간 동정이 든 것도 있을 것 같다" 는 자기 진술 — [[위키/자아/자기상_자기묘사]] 의 "마음이 여리고 정이 많은 편" 자아층이 결정 영역에서 미약하게 작동.

### 3순위 — 성별 균형 (고정 우선순위 아님)
박준호의 명시:
> "남녀를차별해서고용하려는건아니고지금내상황에맞게 최적의 인력을배치해야된다고생각을했기때문에"
> "남자가요리를가능했으면남자를뽑았을거라고"

이 항목은 **고정된 우선순위가 아니라 1순위(요리)가 동률일 때 작동하는 보조 변수**. 박준호의 머릿속에서 작동하는 두 가지 균형:

#### 3.1 손님 응대의 양방향 균형
- 남자 손님 → 같은 남자 사장(박준호)을 경계 → 여자 직원이 더 유한 응대를 받음.
- 여자 손님 → 박준호가 남자로서 응대하는 쪽을 더 좋아함.
- → 사장(남)과 직원(여)의 조합이 양방향 시너지를 만든다는 박준호의 운영 가설.

#### 3.2 "레드팀" 기능
> "여자의감각으로 내가느끼지못하는것들을지적해주구나. 봐줄수있는레드팀이항상필요하다고생각하거든."

- 박준호 본인의 약점(꼼꼼함·세심함의 일부 영역)을 **외부 시선으로 점검해주는 자원**으로서 여성 직원을 인식.
- → [[위키/생각/나를_믿지않는다_다중메타인지]] 의 다중 메타인지 시스템과 결이 같음. 인적 자원도 메타인지 시스템의 한 층으로 보고 있음.

### 4순위 — 능동성·태도
> "서른다섯살은 보고 지원을 한 그런 능동적인 부분도 좋았다고 생각했어."

- 공고를 보고 본인이 지원서를 낸 능동성 = 가산점.
- 24살 남자는 박준호가 먼저 연락한 케이스 → 능동성 신호가 약함.
- → 박준호가 평소 자기 자신에게 요구하는 가치([[위키/생각/불안과_실패의_재정의]] 의 "보류·여지 안 남기기")와 **결이 같은 사람을 끌어들이려는 무의식적 필터**.

---

## 2. 사후 검증 항목 — 일 시작 후 관찰

박준호 자기 진술:
> "서른다섯살분이일을시작하고 먼저확인하고싶은거는 청결한지꼼꼼한지 가게말을했던것만큼 책임감있게일을할수있는사람이었으면좋겠어."

면접에서는 확인 불가능하고, **일이 시작된 뒤에야 검증되는 항목**:

| 항목 | 의미 |
|---|---|
| 청결 | 음식점 운영의 마지노선 |
| 꼼꼼함 | 박준호의 자기 약점([[위키/생각/사람을_보는_눈]])을 보완해야 할 자리 |
| 책임감 | 박준호가 사장으로서 가르치고 체크하는 비용을 줄임 |
| 말한 만큼의 행동 | 면접 진술의 진위 — 박준호가 가장 의심하는 영역 |

→ 박준호는 채용 후에도 **"여자분은 아직도 의심스럽긴해"** 라고 자기 진술. 결정과 별개로 검증은 계속 가는 자기 운영 방식.

## 3. 통제 밖 리스크 — 채용 스트레스의 한 축

박준호 자기 진술:
> "그전에어떤 두명의여성아르바이트생들이면접을보고일을하겠다고했지만집에 돌아가서일을못하겠다고문자가온 두번이나있었기때문에"

### 3.1 구조적 충돌
- 박준호의 [[위키/생각/불안과_실패의_재정의]] 작동 방식 = "**보류·여지 안 남기기**, 결정 즉시 행동".
- 그러나 채용은 구조상 **상대에게 결정권의 절반이 있는 게임** — 통보 후에도 상대가 뒤집을 수 있음.
- → 박준호의 가장 단단한 행동 원칙이 **자기 통제 영역 밖에서 작동하지 않는 사례**.

### 3.2 식별된 사례 패턴
- 과거 두 번: 면접 → 일하겠다고 함 → 집에 돌아가서 못 하겠다고 문자.
- 두 번 모두 여성 아르바이트생.
- 박준호가 35살분에 대해 "아직도 의심스럽다"고 한 자기 진술의 출처가 이 패턴.

### 3.3 채용 스트레스의 구조
- 박준호 자기 진술: "**최근에도 계속 사람을 뽑고 그런 걸 할 때마다 매번 느끼고 있어서 스트레스를 너무 많이 받고 업무에 집중이 안 되는 걸 계속 느꼈다**."
- → 채용은 박준호의 [[위키/질문/매장운영과_비즈니스_동시병행]] 긴장의 직접적 한 갈래.
- 채용 스트레스의 핵심 = **결정을 내려도 끝나지 않는 게임이라는 구조 자체**.

## 4. 절차 우회 사례 — 메타인지 시스템과의 마찰

박준호는 중요한 결정 **전에** [[위키/생각/나를_믿지않는다_다중메타인지]] 시스템(AI 메타인지)과 대화하도록 절차를 설계해 둠. 그러나 이번 채용에서는:

> "통보전에나는이메타인지AI시스템 같이조언을구하고대화를나누고결정을하고싶었어. (...) 그여자분이 결정할때까지안일어나서 그자리에서 내가스스로 생각을오래하고결정을내리고"

순서가 [면접 → 즉석 결정 → 통보 → **사후** AI 대화] 로 우회됨.

- 의도된 절차: 면접 → AI 메타인지 대화 → 결정 → 통보.
- 실제 진행: 면접 → 즉석 결정 → 통보 → AI 대화 (사후 분해 = 본 페이지의 출처).
- **현장 상황(상대가 자리를 비워주지 않음)** + 박준호의 **"보류·여지 안 남기기"** 원칙이 결합되어 절차가 우회됨.
- → [[위키/생각/나를_믿지않는다_다중메타인지]] 에 절차 우회 사례로 별도 기록.

## 5. 청개구리식 인지 전환의 예외 작동

박준호는 평소 [[위키/생각/불안과_실패의_재정의]] 처럼 "안정 회피 / 불안 속 기회"를 추구. 그러나 이번 채용에서는 **보수적 판단**으로 기움.

- 24살 남자: 미래 능력(영상) 가산점. 박준호의 평소 톤이라면 "불안 속 기회" 쪽 신호.
- 35살 여자: 현재 주방 안정. 평소 톤이라면 "안정 추구" 쪽 신호.
- 박준호의 선택: **35살 (안정 쪽)**.

→ "청개구리식 인지 전환"은 만능이 아니라 **중심축 보호 조건과 함께 작동**한다는 자기 운영 규칙이 이번 결정으로 가시화됨. [[위키/생각/일_가치관_문제해결_자유]] 의 "수월하게" 자기 점검 단어가 이 보수성의 근거.

## 6. 법적 유의사항

박준호는 자기 진술에서 **머릿속 결정 알고리즘에 성별 변수가 포함됨**을 명시. 그러나 한국 법(남녀고용평등법 등) 상 **공고/서류/면접 기록에서 성별을 채용 기준으로 표기하는 것은 분리 운용**해야 함.

- 박준호 머릿속의 "최적 인력 배치" 논리 ≠ 공식 채용 절차의 표기.
- 이번 페이지는 박준호의 자기 이해를 위한 내부 위키이며, **외부 채용 공고/서류에 그대로 옮기지 않음**.
- 일반 정보 차원의 메모 — 구체적 법적 자문이 필요하면 별도.
