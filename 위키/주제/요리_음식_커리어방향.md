# 요리 / 음식 / 커리어 방향

박준호의 인생을 관통하는 가장 큰 **반복 패턴**이자 **현재 매일의 일**. 처음 만나는 사람에게 자기를 소개할 때 가장 먼저 꺼내는 정체성("요리덕후") 이 추상이 아니라 — **이미 가게(아재꼬막)를 운영하면서, 양념장 B2B 사업화, 신메뉴 R&D, 새 사업(초콜릿/싸는음식) 까지 동시에 굴리는 실제 행동** 으로 발현되어 있다.

---

## 1. 세 가지 증거의 정렬

박준호의 진로가 음식·요리로 향하고 있다는 사실은 **세 개의 독립적 데이터**가 동시에 한 방향을 가리켜 확증된다.

### 증거 1 — 자기 정체성
- 자기 묘사 첫 단어: "**요리덕후**"
- 처음 만난 사람에게 자기를 소개할 때 가장 먼저 꺼내는 정체성.

### 증거 2 — 학력
- 한양사이버대학교 **호텔외식경영학과** 4학년 마지막 학기.
- 토목 → 전기에서 완전히 트랙을 바꿔 도달한 학과.

### 증거 3 — 자격증
- **식품가공기능사** 필기 합격, 실기 예정.
- 학력이 이론·경영이라면 자격증은 현장·기능. **두 축을 동시에 쌓는 중**.

→ 세 데이터가 우연히 같은 방향을 가리킬 확률은 낮다. **박준호의 진로 방향성은 식음·요리 영역으로 명확히 정렬돼 있음**.

---

## 2. "요리덕후"라는 단어의 무게

박준호가 "요리하는 사람"이 아니라 "**요리덕후**"라고 자기를 부른다는 점이 중요하다.

- "덕후"는 단순히 잘하는 사람이 아닌 **깊이 빠진 사람**.
- 효율·생계를 넘어 **그 자체의 즐거움을 위해** 요리한다는 함의.
- ENTJ의 Te(효율) 측면을 넘어 **INFP Ranger의 의미·미감 영역**이 작동하고 있음을 시사.

→ 즉, 요리는 박준호에게 **두 자아가 만나는 지점**:
- Crusader: 외식 경영·식품가공이라는 **현실적 산업화 가능 트랙**.
- Ranger: 음식·맛·과정에 대한 **개인적 의미·미감**.

이 두 가지가 한 영역에서 맞물리는 것은 박준호의 인생에서 흔치 않은 일. **이게 진짜 길일 가능성이 높은 이유.**

---

## 3. 박준호의 음식 영역 — 2차 인터뷰로 확정

2차 인터뷰에서 박준호의 음식·요리 정체성이 매우 구체적으로 드러났다.

### 3.1 운영 중인 가게
- **아재꼬막** — 박준호가 운영 중인 외식 매장. 생계의 베이스.
- 페이지: [[위키/주제/아재꼬막_가게]]
- 가게 이름이 광주 식문화(꼬막)와 친근한 호칭(아재) 의 결합 — **광주 + 한식 + 정 많은 자기상**의 정확한 표현.

### 3.2 신메뉴 R&D
박준호 자기 진술:
> "요리는 요새 **신메뉴 개발 중인데 육회를 어떻게 만들지에 대한 고민**이 많고, **전통적인 한식과 우리가 어릴 때 먹던 잊혀져 가던 한식**에 대해 고민이 많아."

→ **두 개의 R&D 축**:
1. **육회** — 어떻게 만들지에 대한 구체적 고민. 가게의 시그니처 후보 가능성.
2. **잊혀진 한식** — 전통 한식 + 어릴 때 먹던 잊혀져 가는 한식. 단순한 복고가 아닌 **기억의 복원**을 다루는 영역.

### 3.3 새 사업 후보
박준호 자기 진술:
> "**새로운 사업으로 초콜릿 사업**에 대해, 그리고 **만두나 김밥 같이 어떤 거를 싸서 만드는 형태의 음식**들에 빠져있어. 초콜릿도 그 중 하나이고."

→ "**싸는 음식**" 이라는 박준호만의 카테고리.
→ 페이지: [[위키/주제/싸는음식_초콜릿_새사업]]

### 3.4 사업화 트랙
- **양념장 B2B** — 가게에서 출발해 식품 제조·공급으로 확장.
- 페이지: [[위키/주제/양념장_B2B_사업]]

### 3.5 종합 음식 지도
| 영역 | 박준호의 위치 |
|---|---|
| 한식 (전통) | 육회·잊혀진 한식 R&D |
| 한식 (가게) | 아재꼬막 운영 |
| 식품 제조 | 양념장 B2B (개발 중) |
| 양식·디저트 | 초콜릿 사업 (발상 단계) |
| 싸는 음식 카테고리 | 만두/김밥/초콜릿을 한 묶음으로 본 자기 정의 |
| 푸드 콘텐츠 | (현재 외) |
| 파인다이닝 | (현재 외) |

→ 박준호는 **한식을 중심에 두고 식품 제조·디저트·새 카테고리 정의** 까지 확장하는 외식 사업가의 길을 걷고 있다. 단순 셰프가 아닌 **기획자·운영자·발명가**.

## 4. 진로 — 2차 인터뷰로 확정된 그림

기존 페이지에서 **시나리오 A/B/C/D 중 어디로?** 라고 물었지만, 2차 인터뷰로 답이 명확해짐:

> 박준호는 **시나리오 A(외식 운영) + B(식품 제조) 를 동시에**, 그리고 **이를 법인화해서 투자 유치하는 사업가 트랙**을 이미 진행 중.

```
[아재꼬막 매장 운영]  ← 현재 (시나리오 A 실현)
       +
[양념장 B2B 사업]  ← 진행 중 (시나리오 B 실현)
       +
[초콜릿/싸는음식]  ← 발상 단계
       +
[법인 전환 + 투자 유치]  ← 운영 모델
       +
[AI 에이전트로 회사 자동화]  ← 인프라
       +
[경희대 대학원 진학]  ← 학술 보강
```

→ 호텔외식경영 학과 자체는 **석사 진학 자격을 채우기 위한 단계**. 박준호의 진짜 그림은 **외식 사업가 + 식품 기업가**.
→ 페이지: [[위키/질문/경희대_대학원_진학]]

### 호텔외식경영에 대한 박준호의 솔직한 마음
> "호텔외식경영은 석사를 위해 관련 학과를 지원해서 다닌 거라 큰 생각이 없어."

→ **학사는 사다리, 석사가 진짜 도구**. 학사 5년이 헛된 것이 아니라 사다리의 한 칸일 뿐.

## 5. 토목·전기에서 외식으로 — 트랙 변경의 의미

박준호의 진로 트랙 변경(공학 → 외식)이 단순히 "안 맞아서 바꿈"이 아닐 가능성:

### 가능한 해석들
1. **현장→감각**: 토목·전기는 추상적 시스템. 외식은 **손·맛·사람**의 즉각성이 있는 영역. 박준호의 INFP Ranger 측면에 더 잘 맞는 영역으로 옮긴 것.
2. **타인 영향**: 가족 중 외식·음식 종사자가 있을 가능성 (추후 확인).
3. **결정적 경험**: 어느 시점에 어떤 음식·어떤 식당·어떤 사람이 결정적이었을 수 있음 (추후 확인).
4. **자기 안의 항상 있던 길**: 어릴 때부터 요리에 끌렸으나, 청년기 진학 시점에는 "공학이 안정적"이라는 외부 압력에 따라갔다가 결국 자기 본래로 돌아온 케이스.

## 6. 광주와 음식

(추정) 광주는 음식 문화가 강한 도시. 박준호가 "정신적 고향"이라 부르는 광주에서 보낸 19년이 미각·식문화의 기초를 형성했을 가능성.

→ 추후 인터뷰: 광주의 어떤 음식·식당·식문화가 박준호의 요리관에 영향을 줬는지.
