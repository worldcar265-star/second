# 향미유 R&D 의뢰 — 2026 Q2

프랜차이즈 유통사로부터 받은 향미유 3종 R&D 의뢰 케이스. azae 컨설팅 사업모델의 첫 실전 검증 케이스.

## 의뢰 개요

- **의뢰처**: 프랜차이즈 유통사 (구체 명칭 미공개)
- **의뢰 내용**: 라유 · 치유 · 양파 범용 향미유 3종 R&D
- **의뢰처가 제공한 옵션**: ①공장 리스트업만 ②제조 납품까지
- **변경 동기**: 기존 OEM 단가 비쌈, 단가 절감 + 품질 검토
- **금요일 미팅 공개 예정**: 월 사용량

## 양파 범용 향미유 발상

박준호 통찰: 범용 향미유 = 양파 향미유로 설계 시 의뢰처가 별도 의뢰한 미역국·소고기무국 향미유까지 한 번에 해결 가능.

- 라멘: 양파 향미유 베이스
- 미역국·소고기무국: 양파 향미유 그대로
- → RnD 1종 = 메뉴 3가지 커버

## 베이스 + 향미팩 시스템 발상

박준호 통찰: 액상 향미유 1~2종 + 건조 향미팩 N종 시스템.

- 양파 베이스 + 라유팩 → 라유
- 양파 베이스 + 마늘·생강팩 → 마늘생강 향미유
- 치유는 별도 액상 유지 (닭기름 풍미 재현 어려움)

향신료 vs 향신채 구분 (박준호 핵심 통찰):
- 향신료(산초·계피·후추 등) → 가루팩 OK
- 향신채(마늘·생강·고추 등) → 액상 또는 동결건조 분말 필요

## 미팅 사전 합의 사항

- 타당성 분석은 **무료 상담**으로 진행
- 본 R&D 착수는 **유료** (업계 표준: 1종 500만원 / 3종 약 800만원)
- 박준호가 클로드 켜놓고 실시간 시뮬레이션

## 미팅 시 받아야 할 정보 6개

1. 월 사용량 (종별)
2. 현재 OEM 단가
3. 현재 사용 중인 향미유 종류·스펙
4. 변경 동기 (단가만? 품질도? 공급 안정성?)
5. 납품 형태 희망 (자기들 브랜드? PB? OEM 공장 직거래?)
6. 매장 수

## 미팅 시 제시할 옵션

- **옵션 A**: 비교견적 소싱 (공장 매칭만)
- **옵션 B**: 레시피 개발 + OEM 매칭 (개발비 + 로열티)
- **옵션 C**: 멜트컴퍼니 도매 공급 (제조 납품)
- **옵션 D**: 매장 제조 베이스 + 향미팩 시스템 (박준호 독창 솔루션)
