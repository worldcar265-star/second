# 향미유 R&D 의뢰 — 2026 Q2 (늘프레시 케이스)

**늘프레시**(프랜차이즈 유통사)로부터 받은 향미유 3종 R&D 의뢰 케이스. azae 컨설팅 사업모델의 첫 실전 검증 케이스이자, BNI 네트워킹 → 음식 신뢰 → 의뢰 수주로 이어진 박준호의 첫 외부 영업 검증 사례.

## 의뢰 개요

- **의뢰처**: **늘프레시** (프랜차이즈 유통사)
- **대표이사**: [[위키/사람들/채상훈_늘프레시]] — 최종 의사결정자
- **추천·중개자**: [[위키/사람들/오재석_늘프레시]] — BNI 멤버, 박준호 음식 신뢰 형성 후 의뢰 추천
- **의뢰 내용**: 라유 · 치유 · 양파 범용 향미유 3종 R&D
- **의뢰처가 제공한 옵션**: ①공장 리스트업만 ②제조 납품까지
- **변경 동기**: 기존 OEM 단가 비쌈, 단가 절감 + 품질 검토
- **금요일 미팅 공개 예정**: 월 사용량

## 관계 형성 배경 — BNI → 음식 신뢰 → 의뢰 추천

박준호의 R&D 의뢰 수주는 추상적 영업의 결과가 아니라 **실제 음식 제공·시식을 통한 신뢰 형성**의 결과:

```
박준호 ──[BNI 모임 케이터링 직접 준비]── 오재석 대표
       ──[데리야끼 v3.3 시식 제공]──
                                      ↓ [음식 신뢰 형성]
                                  [향미유 R&D 의뢰 추천]
                                      ↓
                                  채상훈 대표 (최종 결정)
                                      ↓
                                  향미유 R&D 의뢰 발주
```

### 신뢰 자산이 된 두 사건
1. **BNI 조찬 케이터링** — 박준호가 BNI 모임용 조찬 직접 준비·제공 ([[위키/주제/BNI_조찬모임_네트워킹]])
2. **데리야끼 소스 v3.3 시식** — 박준호 R&D 자산 ([[위키/주제/소스_RnD_진데리야끼]]) 시식 제공

→ 박준호의 [[위키/질문/사람과의_시간_부족함]]에서 표현된 "사람과의 시간을 늘리고 싶다"가 사업과 자연스럽게 결합된 사례.

### 음식 신뢰의 비대칭
- **오재석 대표**: 박준호 음식 시식 경험 **있음** → 강한 음식 신뢰
- **채상훈 대표**: 박준호 음식 시식 경험 **없을 가능성** → 음식 신뢰 약함, 단가·비즈니스 변수 중심 판단 추정

→ 금요일 미팅에서 채상훈 대표 참석 시 추가 시식·시연 카드 검토 필요.

## 양파 범용 향미유 발상

박준호 통찰: 범용 향미유 = 양파 향미유로 설계 시 의뢰처가 별도 의뢰한 미역국·소고기무국 향미유까지 한 번에 해결 가능.

- 라멘: 양파 향미유 베이스
- 미역국·소고기무국: 양파 향미유 그대로
- → RnD 1종 = 메뉴 3가지 커버

## 베이스 + 향미팩 시스템 발상

박준호 통찰: 액상 향미유 1~2종 + 건조 향미팩 N종 시스템.

- 양파 베이스 + 라유팩 → 라유
- 양파 베이스 + 마늘·생강팩 → 마늘생강 향미유
- 치유는 별도 액상 유지 (닭기름 풍미 재현 어려움)

향신료 vs 향신채 구분 (박준호 핵심 통찰):
- 향신료(산초·계피·후추 등) → 가루팩 OK
- 향신채(마늘·생강·고추 등) → 액상 또는 동결건조 분말 필요

## 미팅 사전 합의 사항

- 타당성 분석은 **무료 상담**으로 진행
- 본 R&D 착수는 **유료** (업계 표준: 1종 500만원 / 3종 약 800만원)
- 박준호가 클로드 켜놓고 실시간 시뮬레이션

## 미팅 시 받아야 할 정보 6개

1. 월 사용량 (종별)
2. 현재 OEM 단가
3. 현재 사용 중인 향미유 종류·스펙
4. 변경 동기 (단가만? 품질도? 공급 안정성?)
5. 납품 형태 희망 (자기들 브랜드? PB? OEM 공장 직거래?)
6. 매장 수

## 미팅 시 제시할 옵션

- **옵션 A**: 비교견적 소싱 (공장 매칭만)
- **옵션 B**: 레시피 개발 + OEM 매칭 (개발비 + 로열티)
- **옵션 C**: 멜트컴퍼니 도매 공급 (제조 납품)
- **옵션 D**: 매장 제조 베이스 + 향미팩 시스템 (박준호 독창 솔루션)
